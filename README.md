# BookBundler

중고책 여러 권을 살 때, 배송비 포함 총비용이 가장 싼 판매자 조합을 자동으로 찾아주는 CLI 도구.

알라딘과 YES24의 중고 매물을 동시에 검색하고, 같은 판매자에서 묶어 사면 배송비를 아낄 수 있는 최적 조합을 MIP(혼합정수계획법) 솔버로 계산합니다.

## 왜 필요한가

중고책을 온라인으로 여러 권 살 때, 각 책의 최저가만 보고 따로 사면 배송비가 책마다 붙어서 비쌉니다. 같은 판매자에서 묶어 사면 배송비를 한 번만 내니까 더 싸질 수 있는데, 수십 명의 판매자와 수백 개의 매물을 조합해서 최적을 찾는 건 사람이 할 일이 아닙니다.

BookBundler가 대신 합니다.

## 설치

```bash
git clone https://github.com/SeongJaeSong/bookbundler.git
cd bookbundler
pip install -e .
```

Python 3.10 이상이 필요합니다.

## 사용법

### 기본 사용

```bash
# 쉼표로 구분 (따옴표 불필요)
bookbundler search 데미안,노인과 바다,이방인

# 따옴표로 구분
bookbundler search "데미안" "노인과 바다" "이방인"
```

### ISBN으로 검색

제목이 겹칠 수 있는 책은 ISBN으로 정확하게 지정할 수 있습니다. 10자리, 13자리 모두 지원.

```bash
bookbundler search 9788937460470,노인과 바다,이방인
```

### 옵션

```bash
# 특정 플랫폼만 검색
bookbundler search -p aladin 데미안,이방인
bookbundler search -p yes24 데미안,이방인

# 상태 필터
bookbundler search -c 최상 데미안,이방인
bookbundler search -c 상 데미안,이방인
```

## 동작 방식

1. 알라딘/YES24에서 각 책의 중고 매물을 스크래핑 (여러 페이지, 알라딘 중고매장 포함)
2. 각 매물의 판매자, 가격, 상태, 배송비, 무료배송 임계값을 수집
3. PuLP(CBC 솔버)로 배송비 포함 총비용을 최소화하는 판매자 조합을 계산
   - 같은 판매자에서 묶으면 배송비 1회
   - 무료배송 임계값 초과 시 배송비 0원 처리
   - 가격 차이 1,000원 이내면 좋은 상태(최상 > 상 > 중 > 하) 우선
4. 주문별로 판매자, 소계, 링크를 포함한 구매 가이드 출력

## 기술 스택

- **httpx + BeautifulSoup4** - 웹 스크래핑
- **PuLP (CBC 솔버)** - 조합 최적화
- **click** - CLI
- **rich** - 터미널 출력

## 주의사항

- 개인 용도의 도구입니다. 대량 요청은 사이트에 부담을 줄 수 있으니 상식적인 범위에서 사용해주세요.
- 중고 매물은 실시간으로 변동됩니다. 검색 결과를 확인한 뒤 빠르게 구매하는 것을 권장합니다.
- 사이트 구조가 변경되면 스크래퍼가 동작하지 않을 수 있습니다.
